우대이율(연%p, 세전)
  • - 우대이율은 가입당시 농·축협별로 고시한 우대이율을 적용
  • - 아래 우대요건을 충족하고 만기해지하는 경우, 아래에서 정한 우대이율을 각각 적용하며 기본이율과 합산 하여 제공
  • 우대이율(연%p, 세전) 항목의 우대이율 설명 테이블
    구분 우대조건 최고 우대금리(%p, 세전, 연이율) 최저 우대금리(%p, 세전, 연이율)
    선택옵션주1) 가입금액주2) 주머니 ‘백만원’ 단위 선택 시 최고 0.1 이내 0.01
    주머니 ‘천만원’ 단위 선택 시 최고 0.2 이내 0.02
    주머니 ‘억원’ 단위 선택 시 최고 0.3 이내 0.03
    우대쿠폰1 주머니적금 또는 주머니정기적금 동시가입시주3) 최고 0.2 이내 0.01
    우대쿠폰2 2030세대 가입 시주4) 최고 0.1 이내 0.01
    우대쿠폰3 주머니 “백만원” 또는 “천만원” 단위 가입 시주5) 최고 0.1 이내 0.01
    우대쿠폰4 우리 농축협에서만 더 드리는 보너스금리 최고 0.2 이내 0.01
    우대쿠폰5 농축협 첫거래 고객 시 최고 0.1 이내 0.01
  • 주1. 각 주머니는 금액단위별로 백만원단위, 천만원단위, 억원단위로 나뉘어지고 우대금리는 각 금액단위별로 차등적용되며, 금액단위별로 계좌가 개설됨
  • 주2. 각 주머니 금액단위별 가입금액 선택
  • 주3. ‘동시 가입’이란 본 상품 가입 이후 만기 전전월 말일까지 해당 교차구매상품의 가입이력이 있으면 인정되며 관리점은 본 상품 및 교차구매상품이 동일(본점 기준)하여야 함
  • 주4. 가입일 기준 가입자 연령이 만20세 이상~만39세 이하까지 해당
  • 주5. 주머니 “천만원” 또는 “억원” 단위 구매 시, 각각 계좌별로 우대금리 추가 적용
