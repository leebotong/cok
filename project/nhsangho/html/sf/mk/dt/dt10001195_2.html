우대금리 (연%p, 세전)
  • - 아래 우대요건을 충족하고 만기해지하는 경우 기본이율에 가산하여 아래에서 정한 우대이율을 합산 적용
  • - 우대이율의 실제 적용이율은 해당 농·축협별로 다르게 적용될 수 있으므로 가입 전 필히 농·축협별로 확인
  • 우대금리
    우대조건 우대금리
    (연%p, 세전)
    고향사랑기부금 납부고객 우대 0.01 ~ 0.5
    만 65세 이상 고령자 우대 0.01 ~ 0.1
    농축협별 자체 우대금리 0 ~ 1.0
[고향사항기부금 납부고객 우대]
  • - 이 우대금리는 이 적금을 가입한 해당연도 1월1일부터 이 적금의 만기일이 속한 달(M)의 전전월(M-2) 말일까지 고향사랑기부금 납부실적이 확인되는 경우 적용
  • - 우리 농협을 비롯한 다른 농·축협(「농업협동조합법」에 따른 농업협동조합중앙회의 회원인 조합) 또는 농협은행 영업점 창구에서 납부한 것 이외의 다른 방식으로 기부금을 납부한 경우에는 가입고객이 영업점에 증빙서류(www.ilovegohyang.go.kr에서 발급한 납부확인증 또는 영수증 등)를 제출하는 경우에 한하여 우대금리를 적용
[만 65세 이상 고령자 우대]

- 이 적금 가입 고객 중 가입한 해당연도에 만 65세 이상 되는 사람(만 65세가 되는 해당연도의 12월31일까지 이 적금을 가입 시 해당)이 해당
→ 적용 가능 최종연도 = 상품가입연도 – 65
예) 2023년 가입시 적용 가능 고객 : 생년월일 1958.12.31.까지인 사람
