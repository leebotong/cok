재환전(되팔기)이란?

  • 환전신청 후 정정이나 취소는 원칙적으로 불가합니다.(미 수령시, 수령예정일로부터 8영업일째 되는 날 1회차 매매기준율로 자동 재환전(되팔기)되어 고객님의 당초 원화계좌로 입금 됩니다.
  • 다만, 해당 메뉴는 고객님 편의를 위해 당초 재환전(되팔기) 예정일보다 빠르게 재환전(되팔기) 신청 가능하도록 도와드리는 메뉴입니다.

재환전(되팔기) 시 유의사항

  • 신청하신 환전 건 중 미수령 건만 재환전(되팔기)신청이 가능합니다(일부 금액 재환전은 불가)
  • 재환전(되팔기) 신청 후 익 영업일에 고객님의 당초 원화출금 계좌로 입금되며, 적용환율은 신청일 익영업일의 1회차 매매기준율입니다.
  • 당초 원화출금계좌 해지 등의 이유로 계좌가 존재하지 않으면 입금이 불가합니다.
  • 재환전(되팔기) 신청완료 후에는 취소가 불가하오니, 신청 시 유의바랍니다.
