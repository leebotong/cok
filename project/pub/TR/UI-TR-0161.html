  • 매일 설정하신 스윙 금액 이하의 금액이
    적금계좌로 자동저금(자동이체)됩니다.

스윙 기능 사용

스윙금액
스윙금액
  • 출금계좌에서 자동이체 될 금액(스윙 금액)을 설정합니다.
  • 스윙 금액의 단위는 자유롭게 선택할 수 있습니다. 예를 들어 스윙 금액을 2천원 미만으로 선택한 경우 연계된 통장의 매일 잔액 중 1,000 ~ 1,999원까지 본 적금으로 자동이체 됩니다.
  • 스윙 금액 단위변경은 적금 만기일 전, 매 영업일(공유일 제외)09시~18시에 가능합니다.
확인사항

스윙 금액 변경 시 익일부터 적용됩니다.
