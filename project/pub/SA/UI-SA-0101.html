인증서 폐기를
시작합니다.

  • 인증서 폐기 전 확인사항
    금융인증서 폐기를 위해서는 전자금융 가입과 보안매체 발급이 반드시 필요합니다.

    인증서 폐기 유의사항 확인

    • 다른 은행에서 발급한 금융인증서는 해당 은행에서만 폐기할 수 있습니다.
    • 금융인증서 폐기 완료 후, ‘금융인증서 삭제’ 버튼을 통해 폐기된 금융인증서를 삭제해주시기 바랍니다.

    전자금융 가입확인

    • 가까운 영업점 방문 혹은 NH스마트뱅킹 앱을 통해 전자금융가입이 가능합니다.

    보안매체 발급 확인

    • 콕PASS 또는 평면보안카드, OTP 발급이 필요합니다.
    • 콕PASS발급 후 기기변경,앱 재설치를 하신 경우 유효하지 않은 토큰으로 간주되어 콕PASS 사용이 불가합니다.
    • NH안심보안카드는 사용이 불가합니다.
