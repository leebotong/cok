신규 등록할 OTP정보

  • 종류 농협 카드OTP (010 스마트 크리에이티브)
  • 상태 정상
  • 일련번호 1234567890

OTP 이용등록 확인

  • 기존 보안매체 해지가 완료 되었습니다.
  • 기존 보안매체 또는 새로 등록하실 OTP가 사고(분실 등) 신고된 경우 보안매체 발급받으신 금융기관 영업점을 방문하셔야 서비스 이용이 가능합니다.
  • 기존 보안매체를 이용등록 해지하신 경우 새로운 OTP 또는 기존 OTP를 재등록 해야 거래가 가능합니다.
확인사항
  • 미 등록 시 금융거래가 제한될 수 있습니다.
