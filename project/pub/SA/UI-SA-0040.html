OTP 이용등록 안내

  • 농협 또는 타 금융기관에서 발급 받은 OTP를 이용 등록합니다.
  • 보안카드에서 OTP로 전환, 기존 OTP에서 다른 OTP로 전환하실 수 있습니다.

OTP 이용등록 절차

  • 새로 등록할 OTP 상태조회
  • 본인인증 및 기존 보안매체 해지
  • 신규OTP 등록완료
확인사항
  • 기존 보안매체 또는 새로 등록하실 보안매체가 사고(분실 등) 신고된 경우 보안매체를 발급받으신 금융기관 영업점을 방문하셔야 이용이 가능합니다.
  • 보안매체를 해지할 경우 자동 폐기되어 재등록할 수 없습니다.
  • 기존 보안매체를 이용등록 해지하신 경우 새로운 보안매체 또는 기존 보안매체를 재등록해야 거래가 가능합니다.
  • 기존 보안매체를 이용등록 해지하신 경우 새로운 보안매체 또는 기존 보안매체를 재등록해야 거래가 가능합니다.
  • OPT 이용등록은 농·축협 입출금 계좌 등록 후 이용 가능합니다.
