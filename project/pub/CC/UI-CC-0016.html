간편결제 하세요.

제로페이란?

제로페이는 소상공인을 위한 간편결제 서비스로 소비자가 가맹점의 QR코드를 인식하거나, 스마트폰 QR/바코드 제시를 통해 결제가 이루어지는 계좌기반의 직불결제 서비스입니다.
또한, 판매자 계좌로 금액이 이체되는 방식으로 소비자는 현금영수증의 별도 발급없이 연말정산 시 제로페이 이용내역이 자동으로 취합됩니다. 소상공인은 별도 가맹점 수수료 없이 결제대금을 받을 수 있습니다.

소득공제 안내

제로페이 가맹점 결제 시 소득공제율 최대 30%가 적용됩니다.

이용대상자

만 14세 이상 콕뱅크를 가입한 농협은행, 농·축협 입출금계좌 보유 고객

제로페이 결제한도

1일/1회 200만원

QR코드 촬영 결제

  1. ‘QR코드 촬영’을 선택
  2. 매장내 비치된 QR코드를 촬영
  3. 결제금액을 입력
  4. 제로페이 결제 완료

QR/바코드생성 결제

  1. QR/바코드 생성을 선택
  2. 생성된 QR/바코드를 매장직원에게 제시
  3. 매장직원이 QR/바코드 스캔 후 ‘결제확인’ 선택

결제 취소

  1. 제로페이 하단의 ‘결제내역’을 선택
  2. 취소를 원하는 ‘결제내역’을 선택
  3. 상세내역 하단의 거래일련번호(취소코드) 13자리를 매장직원에게 제시
  4. 제로페이 결제 취소완료
