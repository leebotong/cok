모바일현금카드

모바일현금카드는 현금이나 카드없이 휴대폰으로 결제 가능한 서비스로 결제금액의 최대0.5% 캐시백을 드립니다. 가맹점 등록한 하나로마트에서 사용해보세요.

소득공제 안내

모바일현금카드 가맹점에서 결제 시 소득 공제율 최대30%가 신용카드와 합산하여 적용됩니다.

이용대상자

만 14세 이상 콕뱅크를 가입한 농협은행, 농·축협 입출금 계좌 보유 고객

모바일현금카드 결제한도

1일/1회 50만원

QR코드 촬영 결제

  1. ‘QR/바코드 생성’을 선택
  2. 생성된 QR/바코드를 매장직원에게 제시
  3. 매장직원이 QR/바코드 스캔 후 ‘결제확인’ 선택
  4. 모바일현금카드 결제 완료

결제 취소

  1. 모바일현금카드 하단의 ‘결제내역’을 선택
  2. 취소를 원하는 ‘결제내역’을 선택
  3. 상세내역 하단의 취소용 바코드를 매장직원에게 제시
  4. 모바일현금카드 결제 취소완료
