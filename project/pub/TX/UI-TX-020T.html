확인사항
  • 서비스 이용시간: 매일 00:30~23:10
  • 교통범칙금은 가입자 본인내역만 조회/납부 하실 수 있습니다.
  • 고지서에 인자된 “고객관리번호” 또는 “전자납부번호”를 입력하여 주시기 바랍니다.
  • 고객관리번호 <주민 또는 법인(사업자)>로 조회하시면 고객님께 부가된 전체 경찰청 교통범칙금 고지서를 확인 후 납부하실 수 있습니다.
  • 전자납부번호(종이고지서에 표기됨)로 조회하시면 해당 교통범칙금 고지서를 확인 후 납부할 수 있습니다.
