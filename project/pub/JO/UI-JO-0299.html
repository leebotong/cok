    우대이율(연%, 세전)
    • - 우대이율은 가입 당시 해당 농·축협에서 고시한 우대이율 적용
    • 우대요건, 우대이율 적범위, 우대이율 *주2)(연%, 세전)
      우대요건 우대이율 적용범위 우대이율 *주2)(연%, 세전)
      결산기 *주1) 평잔 30만원 이상 일별 잔액 30만원까지 0.01 ~ 0.5
      결산기 평잔 50만원 이상 일별 잔액 50만원까지 0.01 ~ 0.7

    • - 우대금리는 해당 결산기(이자계산일)에 고객이 우대요건을 충족하는 경우, 다음 결산기에 우대금리 적용범위 내에서 우대금리 적용
    • - 우대요건을 해당 결산기 내에 충족하지 못하는 경우, 일별 잔액 전체에 대해 기본금리만 제공. 또한, 우대금리 적용범위를 초과하는 일별 잔액에 대해서도 기본금리만 제공
