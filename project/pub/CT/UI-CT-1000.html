안전한 금융거래를 위해
고객확인 등록을 진행해주세요.

고객확인(KYC) 제도란?

고객확인 등록 준비물
  • 본인명의 휴대전화
  • 신분증(주민등록증, 운전면허증, 모바일운전면허 증 중 1개 선택)
  • 본인명의 농·축협 계좌

고객확인 등록 절차

  1. 본인 휴대전화 인증
  2. 신분증 확인
  3. 농·축협 계좌 인증
  4. 고객정보 확인
  5. 고객확인 등록 완료
확인사항
  • 고객확인 등록은 농·축협 고객만 등록이 가능하며, 농협은행 고객일 경우 스마트(인터넷)뱅킹 또는 영업점을 이용해주시기 바랍니다.
  • 미성년자 및 외국인 고객님께서는 비대면 고객확인 등록을 진행할 수 없으므로 가까운 영업점을 방문해주시기 바랍니다.
